<!-- GETTING STARTED -->


### Prerequisites


* npm
  ```sh
  npm install 
  ```

### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/srinathdevinfo/pegen.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
4. Run
   ```JS
   ng serve or npm start
   ```
